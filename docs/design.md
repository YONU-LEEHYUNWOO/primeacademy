# 🎨 **UI/UX 디자인 가이드: 수학학원 랜딩페이지 (Modern & Professional)**

**작성자:** 시니어 UI/UX 디자이너
**스타일:** Modern
**기준 색상:** `#6fcf1b`
**참고 서비스:** 네이버 (Naver)
**디자인 키워드:** 전문적인 · 신뢰감 · 깔끔함 · 편안한

---

## 📘 1. Design System Overview

이 랜딩페이지의 디자인 시스템은 **‘전문성과 신뢰감을 주는 현대적 UI’**를 목표로 합니다.
“네이버”의 UI처럼 **불필요한 장식은 최소화**하고, **정보의 명료성**과 **UX 흐름의 안정감**을 우선합니다.

### 🎯 디자인 목표

* **전문적이고 신뢰감 있는 시각적 언어 구축**
* **학부모 타깃 UX 설계** (명확한 CTA와 정보 위계)
* **그리드 기반의 안정된 시각 밸런스 유지**
* **편안하고 부드러운 색조 대비**로 시각 피로 최소화

## 🎨 2. Color Palette (TailwindCSS 기반)

### 🌿 아날로그 컬러 스킴 (Analogous Scheme)

기준 색상 `#6fcf1b` (연두) 을 중심으로 인접한 색상군(녹색~황록~청록)을 사용합니다.
이는 **자연스러움·안정감**을 제공하며 학부모 타깃에 적합합니다.

| 컬러 그룹             | 이름              | 코드        | 용도                |
| ----------------- | --------------- | --------- | ----------------- |
| **Primary 500**   | `primary-500`   | `#6fcf1b` | 브랜드 메인 버튼, 강조 포인트 |
| **Primary 700**   | `primary-700`   | `#5ab213` | Hover / Active 상태 |
| **Primary 100**   | `primary-100`   | `#e6f7d5` | 배경형 강조 구역         |
| **Secondary 500** | `secondary-500` | `#56b893` | 보조 강조, 그래픽 포인트    |
| **Accent 500**    | `accent-500`    | `#c5e86c` | 성공 메시지, 하이라이트 요소  |
| **Neutral 900**   | `neutral-900`   | `#1a1a1a` | 주 텍스트             |
| **Neutral 600**   | `neutral-600`   | `#6b7280` | 부 텍스트 및 보조 설명     |
| **Neutral 200**   | `neutral-200`   | `#e5e7eb` | 카드 테두리, 구분선       |
| **White**         | `white`         | `#ffffff` | 배경 및 입력필드         |
| **Error 500**     | `error-500`     | `#e63946` | 입력 오류, 경고 메시지     |

## 🧱 3. Page Implementations (페이지별 디자인 가이드)

### 🏠 **홈 (/)**

**🎯 핵심 목적:**
첫 인상에서 신뢰감과 전문성을 전달하고, 방문자가 “상담 예약” 버튼을 누르도록 유도

**🧩 주요 구성요소:**

* Hero Section (메인 비주얼 + CTA 버튼)
* 학원 소개 섹션 (비전 / 교육 철학)
* 성과 & 통계 (시각화 그래프)
* 차별점 섹션 (3~4 아이콘형 블록)
* 강사진 소개 (사진형 프로필 카드)
* 상담 예약 CTA 영역 (폼 진입 유도)

**🧮 레이아웃 구조:**

* `grid-cols-1 md:grid-cols-2 lg:grid-cols-3`
* Hero 영역: 뷰포트 높이의 80%
* 통계 영역: 차트 중심, 중앙 정렬

**🖼️ 이미지 예시:**
![학원 메인 이미지](https://picsum.photos/1440/600)
(밝은 실내 교실 사진 + 연두색 오버레이로 브랜드 컬러 강화)

### 📈 **성과 통계 페이지 (/results)**

**🎯 목적:**
학원의 객관적인 성과(성적 향상률, 합격률 등)를 시각적으로 제시

**🧩 주요 구성요소:**

* 그래프 (Chart.js or Recharts)
* 비교 카드 (“우리 학원 평균 89% 향상”)
* 후기 섹션 (학부모 평가 요약)

**💡 디자인 포인트:**

* Primary + Secondary 컬러의 그래디언트 사용
* 그래프 라벨 가독성 위해 중간톤(Neutral 600) 사용
* 성공 수치 강조 시 Accent 500 사용

---

### 👩‍🏫 **강사진 소개 페이지 (/teachers)**

**🎯 목적:**
전문성과 신뢰감을 시각적으로 강화하는 페이지

**🧩 주요 구성요소:**

* 강사 프로필 카드
* 수업 철학 / 담당 과목 / 경력
* Hover 시 상세 프로필 표시

**🧱 레이아웃 구조:**

* 데스크톱: 3열 Grid
* 모바일: 1열 Scroll
* 카드: `rounded-2xl shadow-sm border border-neutral-200 p-6`

**🖼️ 이미지 예시:**
![강사 프로필 이미지](https://picsum.photos/400/400)

### 🗺️ **오시는 길 (/location)**

**🎯 목적:**
위치 정보와 접근성을 직관적으로 전달

**🧩 주요 구성요소:**

* Google Maps Embed
* 학원 주소 및 교통편 안내
* 연락처, 운영시간
* 하단 “상담 예약” 버튼

**💡 UI 특징:**

* 지도 위에 브랜드 컬러 Marker
* 연락처 섹션에 아이콘(`📞`, `🕒`) 병행

---

### 📅 **상담 예약 페이지 (/reservation)**

**🎯 목적:**
상담 신청 폼을 통해 전환 유도

**🧩 주요 구성요소:**

* 입력 필드 (이름, 연락처, 날짜, 문의내용)
* 제출 버튼 (`bg-primary-500 hover:bg-primary-700`)
* 성공 시 확인 메시지

**💡 인터랙션:**

* 실시간 유효성 검사
* 오류 시 빨간 테두리(`error-500`)
* 성공 시 “성공” 메시지 (`accent-500`)

## 🖱️ 5. Interaction Patterns

| 패턴                   | 설명                           | 예시                              |
| -------------------- | ---------------------------- | ------------------------------- |
| **Hover Feedback**   | 버튼/카드 hover 시 색상 변화 및 그림자 강조 | CTA 버튼 hover → `bg-primary-700` |
| **Scroll Animation** | 요소 진입 시 fade-up 애니메이션        | 강사진 카드 등장 시                     |
| **Form Validation**  | 입력 필드별 실시간 오류 피드백            | 이름 미입력 시 테두리 빨간색 표시             |
| **Sticky CTA**       | 스크롤 하단에 “상담 예약” 고정 버튼        | `/results`, `/teachers` 하단      |
| **Loading Feedback** | 폼 제출 시 스피너 표시                | “전송 중…” 문구 노출                   |

## 📱 6. Breakpoints (Tailwind 기준)

```scss
$breakpoints: (
  'mobile': 320px,
  'tablet': 768px,
  'desktop': 1024px,
  'wide': 1440px
);
```

| 구분                      | 최소 해상도                    | 주요 변화점 |
| ----------------------- | ------------------------- | ------ |
| **Mobile (≤768px)**     | 단일 컬럼, 메뉴 숨김 → 햄버거 메뉴로 전환 |        |
| **Tablet (768~1024px)** | 2컬럼 레이아웃, CTA 하단 고정       |        |
| **Desktop (≥1024px)**   | 3컬럼 구조, Hero 섹션 좌우 분할     |        |
| **Wide (≥1440px)**      | 최대폭 1440px, 여백 10% 유지     |        |

## ✏️ 7. 디자인 의도 및 원칙

| 항목          | 원칙                       | 설명                |
| ----------- | ------------------------ | ----------------- |
| **색상 전략**   | 자연스럽고 신뢰감 있는 녹색군         | 학부모 대상의 친숙함 강조    |
| **타이포그래피**  | Pretendard / 16px 기준     | 가독성과 안정감 확보       |
| **아이콘 스타일** | Line 아이콘 + Filled 강조형 병행 | 단조로움 방지, 정보 구분 강화 |
| **버튼 형태**   | 둥근 모서리(8px), 그림자 최소화     | 부드러운 인상 제공        |
| **레이아웃**    | 12그리드 기반, 중앙 정렬          | 균형감과 안정적 시각 구성    |
| **이미지 톤**   | 밝은 실내 + 자연광              | “신뢰·편안함” 전달       |

## ✅ 결론

이 디자인 가이드는 **학부모 신뢰 중심의 Modern UI**를 구현하기 위한 구조적, 시각적 원칙을 제공합니다.
네이버형 **깔끔하고 기능 중심적인 톤앤매너**를 유지하면서,
브랜드 컬러 `#6fcf1b`를 중심으로 **편안한 녹색 계열의 아날로그 팔레트**를 활용해
“전문적이면서 따뜻한” 학원 브랜드 이미지를 완성합니다.

> **핵심 요약:**
>
> * 단순하고 명료한 정보 구조
> * 고대비가 아닌 **자연스럽고 편안한 녹색톤 UI**
> * 명확한 CTA와 부드러운 애니메이션으로 **전환 중심 UX 실현**
